# 회원가입 기능 명세

## 개요
아이디와 비밀번호를 입력하여 회원가입한다.

## 화면 구성
- 아이디 입력 필드
- 비밀번호 입력 필드 (6자 이상)
- 회원가입 버튼
- 로그인 페이지 링크

## 동작 흐름

1. 사용자가 아이디, 비밀번호를 입력한 후 회원가입 버튼을 클릭한다.
2. 비밀번호가 6자 미만이면 "비밀번호는 6자 이상이어야 합니다." 알럿을 노출한다.
3. 회원 테이블에서 아이디 중복을 검증한다.
   - **중복**: "이미 사용 중인 아이디입니다." 알럿을 노출한다.
   - **사용 가능**: 회원 테이블에 아이디, 비밀번호(암호화)를 저장하고 자동 로그인 후 My 페이지로 이동한다.
