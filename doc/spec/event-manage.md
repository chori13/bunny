# 이벤트 관리 기능 명세

## 개요
관리자가 이벤트를 등록, 수정, 삭제한다.

## 접근 경로
- 마이페이지 → "이벤트 관리" 버튼 클릭
- URL: `/events/manage`

## 권한
- 관리자(ADMIN)만 접근 가능
- 비관리자 접근 시 마이페이지로 리다이렉트

## 화면 구성
- 마이페이지 돌아가기 링크
- "+ 이벤트 등록" 버튼
- 등록/수정 폼 (토글)
  - 제목 (필수), 아이콘 (필수, 이모지)
  - 설명 (필수, textarea)
  - 기간 (필수), 상태 (진행중/예정/종료), 색상 (선택)
  - 취소 / 등록(수정) 버튼
- 이벤트 목록 (아이콘, 제목, 상태 배지, 설명, 기간)
  - 수정 / 삭제 버튼

## 동작 흐름

### 등록
1. "+ 이벤트 등록" 버튼 클릭 시 등록 폼이 노출된다.
2. 필수 항목(제목, 설명, 기간, 아이콘)이 비어있으면 "필수 항목을 입력해주세요." 알럿을 노출한다.
3. 등록 성공 시 "이벤트가 등록되었습니다." 알럿 후 목록을 갱신한다.

### 수정
1. 이벤트 목록의 "수정" 버튼 클릭 시 기존 데이터가 채워진 수정 폼이 노출된다.
2. 수정 성공 시 "이벤트가 수정되었습니다." 알럿 후 목록을 갱신한다.

### 삭제
1. "삭제" 버튼 클릭 시 확인 알럿을 노출한다.
2. 확인 시 삭제 처리 후 "이벤트가 삭제되었습니다." 알럿을 노출한다.
