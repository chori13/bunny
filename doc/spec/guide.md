# 건강 가이드 기능 명세

## 개요
토끼 건강 관리에 필요한 정보를 카테고리별로 제공한다.

## 접근 경로
- 헤더 → "건강 가이드" 클릭
- URL: `/guide`

## 화면 구성
- 페이지 헤더 (제목, 설명)
- 4개 카테고리 카드:
  - 식이 관리 (건초/사료/간식 가이드)
  - 생활 환경 (케이지/온도/습도)
  - 건강 체크 (체중/이빨/배변)
  - 그루밍 (빗질/발톱/목욕)
- 각 카테고리별 상세 가이드 항목
- 응급 상황 체크리스트

## 관련 파일
