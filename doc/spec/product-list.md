# 상품 목록 기능 명세

## 개요
관리자가 등록한 상품을 목록으로 조회한다. 관리자는 수정/삭제, 일반 회원은 구매하기 버튼이 노출된다.

## 접근 경로
- 헤더 → "전체 상품" 클릭
- URL: `/products`

## 화면 구성
- 페이지 제목 및 설명
- 관리자: "+ 상품 등록" 버튼
- 상품 카드 그리드 (1~4열 반응형)
  - 이미지, 상품명, 가격, 비고
  - 관리자: 수정 / 삭제 버튼
  - 일반 회원: 구매하기 버튼

## 동작 흐름

1. 페이지 진입 시 상품 테이블에서 전체 상품을 최신순으로 조회하여 카드 형태로 표시한다.
2. 등록된 상품이 없으면 "등록된 상품이 없습니다." 메시지를 노출한다.
3. 상품 카드 클릭 시 상품 상세 페이지(`/products/[id]`)로 이동한다.
4. **관리자 기능**:
   - 수정 버튼 클릭 → 상품 수정 페이지(`/products/[id]/edit`)로 이동
   - 삭제 버튼 클릭 → 확인 알럿 후 삭제 처리
5. **일반 회원**: 구매하기 버튼 노출
