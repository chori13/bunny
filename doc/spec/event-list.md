# 이벤트 목록 기능 명세

## 개요
진행중, 예정, 종료된 이벤트를 상태별로 구분하여 표시한다.

## 접근 경로
- 헤더 → "이벤트" 클릭
- URL: `/events`

## 화면 구성
- 페이지 헤더 (제목, 설명)
- **진행중인 이벤트**: 3열 카드 그리드 (아이콘, 배지, 제목, 설명, 기간, 색상)
- **혜택 배너**: 회원가입/상품 보기 CTA
- **예정된 이벤트**: 2열 카드 그리드
- **종료된 이벤트**: 리스트형 (반투명)

## 동작 흐름

1. 페이지 진입 시 이벤트 API에서 전체 이벤트를 조회한다.
2. status 값(ACTIVE, UPCOMING, ENDED)으로 분류하여 섹션별로 표시한다.
3. 각 상태 그룹에 해당하는 이벤트가 없으면 해당 섹션을 숨긴다.
4. 이벤트가 하나도 없으면 "등록된 이벤트가 없습니다." 메시지를 노출한다.
