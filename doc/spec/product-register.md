# 상품 등록 기능 명세

## 개요
메인 페이지에서 상품 정보를 입력하여 상품을 등록한다.

## 화면 구성
- 상품명 입력 필드
- 첨부파일 입력 필드 (이미지)
- 가격 입력 필드
- 비고 입력 필드
- 상품 등록 버튼

## 동작 흐름

1. 사용자가 상품명, 첨부파일, 가격, 비고를 입력한 후 상품 등록 버튼을 클릭한다.
2. 상품명 또는 가격이 비어있으면 "상품명과 가격을 입력해주세요." 알럿을 노출한다.
3. 첨부파일이 있으면 서버에 이미지를 업로드한다.
4. 상품 테이블에 상품 정보를 저장한다.
5. 등록 성공 시 "상품이 등록되었습니다." 알럿을 노출하고 입력 필드를 초기화한다.

## 관련 파일
- 페이지: `src/app/page.tsx`
- API: `src/app/api/products/route.ts`
- 테이블: [상품 테이블 스키마](../schema/product-table-schema.md)
