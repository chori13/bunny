# 상품 등록 기능 명세

## 개요
관리자가 상품 정보를 입력하여 상품을 등록한다.

## 접근 경로
- 마이페이지 → "상품 등록" 버튼 클릭
- URL: `/products/register`

## 권한
- 관리자(ADMIN)만 접근 가능

## 화면 구성
- 상품명 입력 필드 (필수)
- 첨부파일 입력 필드 (이미지, 선택)
- 가격 입력 필드 (필수)
- 비고 입력 필드 (선택)
- 취소 / 등록 버튼

## 동작 흐름

1. 사용자가 상품명, 첨부파일, 가격, 비고를 입력한 후 등록 버튼을 클릭한다.
2. 상품명 또는 가격이 비어있으면 "상품명과 가격을 입력해주세요." 알럿을 노출한다.
3. 첨부파일이 있으면 서버의 `public/uploads/` 디렉토리에 이미지를 업로드한다.
   - 파일명은 `{timestamp}_{원본파일명}` 형식으로 저장한다.
4. 상품 테이블에 상품 정보를 저장한다.
5. 등록 성공 시 "상품이 등록되었습니다." 알럿을 노출하고 상품 목록으로 이동한다.

## 관련 파일
- 페이지: `src/app/products/register/page.tsx`
- API: `src/app/api/products/route.ts` (POST)
- 테이블: [상품 테이블 스키마](../schema/product-table-schema.md)
