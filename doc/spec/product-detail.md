# 상품 상세 기능 명세

## 개요
상품의 상세 정보를 조회한다. 관리자는 수정/삭제, 일반 회원은 구매하기 버튼이 노출된다.

## 접근 경로
- 상품 목록에서 상품 카드 클릭
- URL: `/products/[id]`

## 화면 구성
- 뒤로가기 링크 (← 상품 목록)
- 2단 레이아웃 (이미지 / 상세 정보)
  - 좌측: 상품 이미지 (이미지 없을 시 "No Image")
  - 우측:
    - 상품명, 등록일
    - 판매가
    - 상품 설명 (비고)
    - 서비스 배지 (무료 배송, 7일 이내 환불, 정품 보장, 빠른 출고)
    - 관리자: 수정 / 삭제 버튼
    - 일반 회원: 구매하기 버튼

## 동작 흐름

1. 페이지 진입 시 상품 ID로 단건 조회한다.
2. 상품이 존재하지 않으면 "상품을 찾을 수 없습니다." 메시지와 목록 돌아가기 링크를 노출한다.
3. **관리자 기능**:
   - 수정 버튼 클릭 → `/products/[id]/edit`로 이동
   - 삭제 버튼 클릭 → 확인 알럿 → 삭제 후 상품 목록으로 이동
4. **일반 회원**: 구매하기 버튼 노출

## 관련 파일
- 페이지: `src/app/products/[id]/page.tsx`
- API: `src/app/api/products/[id]/route.ts` (GET, DELETE)
