# 마이페이지 기능 명세

## 개요
로그인한 사용자의 계정 정보를 표시하고, 관리 기능에 접근할 수 있다.

## 접근 경로
- 헤더 → "마이페이지" 클릭
- URL: `/mypage`

## 권한
- 로그인 필수 (비로그인 시 로그인 페이지로 리다이렉트)

## 화면 구성
- 사용자 프로필 (아바타, 아이디)
- 상품 등록 버튼 (관리자)
- 이벤트 관리 버튼 (관리자)
- 로그아웃 버튼

## 동작 흐름

1. 비로그인 상태에서 접근 시 `/login`으로 리다이렉트한다.
2. 로그인된 사용자의 아이디를 표시한다.
3. **관리자 기능**:
   - "상품 등록" → `/products/register`로 이동
   - "이벤트 관리" → `/events/manage`로 이동
4. "로그아웃" 클릭 시 세션을 종료하고 로그인 페이지로 이동한다.

## 관련 파일
- 페이지: `src/app/mypage/page.tsx`
- 인증: `src/lib/auth.ts`
- 테이블: [회원 테이블 스키마](../schema/user-table-schema.md)
