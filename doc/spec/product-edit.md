# 상품 수정 기능 명세

## 개요
관리자가 등록된 상품의 정보를 수정한다.

## 접근 경로
- 상품 목록 / 상품 상세 → "수정" 버튼 클릭
- URL: `/products/[id]/edit`

## 권한
- 관리자(ADMIN)만 접근 가능

## 화면 구성
- 상품명 입력 필드 (기존 값 표시, 필수)
- 첨부파일 입력 필드 (현재 이미지 미리보기, 선택)
- 가격 입력 필드 (기존 값 표시, 필수)
- 비고 입력 필드 (기존 값 표시, 선택)
- 취소 / 수정 완료 버튼

## 동작 흐름

1. 페이지 진입 시 상품 ID로 기존 정보를 조회하여 폼에 채운다.
2. 상품명 또는 가격이 비어있으면 "상품명과 가격을 입력해주세요." 알럿을 노출한다.
3. 새 이미지를 선택하면 기존 이미지를 대체한다. 선택하지 않으면 기존 이미지를 유지한다.
4. 수정 성공 시 "상품이 수정되었습니다." 알럿을 노출하고 상품 목록으로 이동한다.
